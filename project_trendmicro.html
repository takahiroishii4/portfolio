January - Present, 2020 • UX research in enterprise product

Cloud One Network Security

[ONGOING PROJECT] Onboarding experience of a new cloud security software

Process Competitive analysis, User research, Site-map, Prototyping
Project type UX internship at Trend Micro Austin

Background

Trend Micro is launching a cloud security product "Cloud One Network Security". Traditionally, the onboarding experience (called "first-time experience" here) is painful in the cybersecurity field, because it requires lots of effort and time to learn the software, configure the environment, deploy the instance, and confirm the performance. This project aims to mitigate these pains.

Project statement

Project goals:

  1. Users can understand the product values clearly and quickly
  2. Users can effectively understand how to activate, customize the configuration, and confirm the success

Proposed flows:
Through internal stakeholder interview, I learned the current state of the first-time experience, and what they envision the future on the three following user flows.

Competitive analysis

To better understand the current state of the first-time, experience in the industry. I compared different competitor's products.

Persona development

Through conversations with potential users and internal stakeholders, I defined five different personas with priorities.

User journey mapping

Through discussions, I created two user journeys: one for the current state, and another for the ideal future scenario.
